 تفاوت stack و global/static از نظر طول عمر (lifetime):
نوع متغیر	محل ذخیره	شروع زندگی	پایان زندگی	دسترسی
local variable (مثل داخل main)	Stack	وقتی تابع صدا زده می‌شه	وقتی تابع تموم می‌شه (stack frame پاک می‌شه)	فقط داخل همون تابع
static variable (داخل تابع)	Data segment	وقتی برنامه اجرا می‌شه	تا پایان اجرای برنامه	فقط داخل همون تابع (ولی مقدارش حفظ می‌شه)
global variable	Data segment	وقتی برنامه اجرا می‌شه	تا پایان اجرای برنامه	در همه توابع قابل دسترسی
heap (مثل malloc)	Heap	وقتی malloc یا new صدا زده می‌شه	وقتی free یا delete صدا زده بشه	با استفاده از pointer در هر جایی قابل استفاده است

✅ درباره Stack Frame:
هر بار که تابعی صدا زده می‌شه:

یه Stack Frame برای اون تابع روی Stack ساخته می‌شه.

این Frame شامل:

متغیرهای محلی (local)

return address (RIP)

مقدار قبلی RBP و ...

وقتی تابع تموم می‌شه:

اون Frame پاک می‌شه.

بنابراین، متغیرهای محلی از بین می‌رن.

📌 یعنی Stack فقط موقتیه.
